GHUB NOW

A micro-service, hosted on ZEIT Now Platform, that redirects to correct location of a package. Supports monorepos!

  • /:name
  • /:name@:version
  • /:name@:tag
  • /:name/:branch
  • /@:scope/name
  • /@:scope/name@:version
  • /@:scope/name@:tag
  • /@:scope/name/:branch
